# Cloudflare Pages デプロイ手順

## 1. Cloudflare アカウント作成
https://cloudflare.com でアカウント作成（無料）

## 2. Pages プロジェクト作成
- Dashboard → Pages → Create a project
- Connect to Git または Upload assets

## 3. ビルド設定
- Framework preset: None
- Build command: (空)
- Build output directory: public

## 4. カスタムドメイン
- Pages プロジェクト → Custom domains
- 無料サブドメイン: your-project.pages.dev

## 5. 環境変数設定
Firebase設定はそのまま使用可能
